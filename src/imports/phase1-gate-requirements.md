PHASE 1 READY GATE — BOARDROOM STANDARD

Proposal can move to internal_review only if:

1️⃣ Strategic Integrity Validation

Executive Brief must contain:

Clear quantified operational strain

Clear financial implication

Clear consequence of inaction

Clear transformation outcome

Minimum requirements:

≥ 600 words total across the brief

At least 2 quantified statements (numbers, not adjectives)

No vague phrases like “improve efficiency”

If vague language detected → reject.

2️⃣ Diagnosis Depth Validation

Minimum 3 diagnosis blocks.

Each must include:

Root cause (not symptom)

System-level explanation

Financial translation

Evidence reference

Severity classification

Confidence score ≥ 70

Additionally:

At least 1 diagnosis marked High or Critical

No duplicate financial impact across diagnoses

3️⃣ Financial Alignment Pre-Check

Before ROI even attaches:

Every diagnosis must map to at least one future solution

Estimated financial exposure must be defined (even rough)

No “impact unknown” allowed

If exposure undefined → reject.

4️⃣ Scope Governance Validation

Included scope must:

Be clearly bounded

Contain no implementation ambiguity

Not imply guaranteed financial results

Excluded scope must:

Explicitly block legal/medical/financial advisory

Explicitly block full autonomy AI claims

5️⃣ Executive Readability Score

Internal readability test:

No paragraph longer than 8 lines

No jargon density > 10%

Tone: authoritative, not technical

If reads like engineering doc → reject.

6️⃣ Strategic Coherence Check

System must confirm:

Executive Brief references Diagnosis themes

Diagnosis themes logically justify next-step offer

Offer aligns with severity level

If narrative does not flow logically → reject.

🔒 Gate Output Logic

If ANY check fails:

Status remains draft.

If all pass:

Status becomes internal_review.
