# AI Start Here

Universal bootstrap for every AI agent (Claude, ChatGPT, Codex, Cursor, Gemini).

Before doing any work, read ONLY these three files, in order:

1. `docs/ai/MARQ_CORTEX.md` — rules and standards (highest authority)
2. `docs/ai/ACTIVE_WORK.md` — the current task
3. `docs/ai/CHANGELOG_AI.md` — what is already done

Then:

- Resume from `ACTIVE_WORK.md`.
- `MARQ_CORTEX.md` is the highest authority; no other document may override it.
- Do not re-audit completed work.
- Do not read `docs/ai/archive/` unless explicitly requested.

Reference only when the task requires it: `docs/ai/ROADMAP.md`, `docs/ai/DECISIONS.md`, `docs/ai/STABILIZATION.md`, `docs/ai/EXECUTION_RULES.md`, `docs/ai/TEST_PROTOCOL.md`, `docs/ai/DOCUMENTATION_RULES.md`.
