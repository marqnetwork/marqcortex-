# AI Start Here

Universal entry point for any AI agent (Claude, GPT, Gemini, Cursor, Codex).

Before doing any work, read only these three files, in order:

1. `docs/ai/MARQ_CORTEX.md` — rules and standards (master control)
2. `docs/ai/ACTIVE_WORK.md` — the current task
3. `docs/ai/CHANGELOG_AI.md` — what is already done

Then continue the current task. Never rediscover or restart completed work.

Reference only when needed: `docs/ai/ROADMAP.md`, `docs/ai/DECISIONS.md`, `docs/ai/STABILIZATION.md`, `docs/ai/EXECUTION_RULES.md`, `docs/ai/TEST_PROTOCOL.md`, `docs/ai/DOCUMENTATION_RULES.md`.
